Add Timestamps to Use Cases

⚠️ Important: This script will add today's date as both createdAt and updatedAt timestamps to ALL use cases in your Firebase database that don't already have timestamps.

What this script does:

  • Scans all use cases in your database
  • For cases without timestamps, adds today's date for both createdAt and updatedAt
  • Skips cases that already have timestamps
  • Shows a log of all operations
